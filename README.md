# USDA-ARS Navel Orangeworm Area-wide Project, 2008 to 2012

The USDA Agricultural Research Service funded an Area-wide Project
from 2008 to 2012. This demonstration project examined best practices
for incorporating a recently-registered mating disruption formulation
and newer reduced-risk insecticides with the foundational practices of
orchard sanitation and timely harvest. The following peer-reviewed papers
and patents emerged from this project.
